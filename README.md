# MOZILLA IOT GENERIC SENSORS ADAPTER #


## USAGE: ##

* Install mozilla-iot gateway (0.4.0+) on supported system
* Enable I2C
* It was tested on Raspberry PI 3 along 2 I2C sensors supported by "generic-sensors-lite" module
